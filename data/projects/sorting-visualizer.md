## Algorithms
This application implements a few sorting algorithms which are listed here.
### Merge Sort
Merge sort recursively splits the array until it can not longer. 
Then it will recursively merge these arrays into a sorted result.
### Quick Sort
The quick sort algorithm
### Bubble Sort
The bubble sort algorithm
